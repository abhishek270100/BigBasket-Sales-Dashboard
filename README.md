# BigBasket Sales Dashboard

An end-to-end Business Intelligence project focused on analyzing sales performance and profitability for BigBasket.

![Dashboard Screenshot]
(https://github.com/abhishek270100/BigBasket-Sales-Dashboard/blob/main/Dashboard%20Image.png)

## 📊 Project Overview
The objective of this project was to transform raw sales data into actionable insights. By leveraging Power BI, I created a dynamic dashboard that allows stakeholders to track KPIs, identify top-performing product categories, and analyze payment trends.

## 🚀 Key Performance Indicators (KPIs)
* *Total Sales:* 17K
* *Total Profit:* 2,172
* *Average Order Value (ADV):* 2,637
* *Total Quantity Sold:* 141

## 💡 Key Insights & Findings
* *Payment Trends:* Cash on Delivery (COD) is the dominant payment method (55%), followed by Debit Cards (18%), indicating a preference for offline/immediate payment options.
* *Category Performance:* Clothing accounts for the highest volume of sales (63%), highlighting a clear demand in the apparel sector.
* *Profitability:* Printers are currently the most profitable sub-category, contributing significantly to the overall profit margins.
* *Regional Analysis:* The dashboard allows for filtering by state (e.g., Uttar Pradesh), enabling location-specific sales monitoring.

## 🛠️ Tools & Technologies
* *Data Visualization:* Power BI
* *Data Transformation:* Power Query
* *Calculations:* DAX (Data Analysis Expressions)
